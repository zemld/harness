# Engineering conventions — stack index

Root index for the `read-docs` skill. It lists the stacks that have documented coding conventions. Match the stack you are working in to a row, then open that stack's `index.md` to find the specific doc for your topic.

Available stacks:

- **go** — [go/index.md](./go/index.md) — Go services: layout, style, testing, dependencies.
- **frontend** — [frontend/index.md](./frontend/index.md) — React/TypeScript apps: structure, style, state, forms, API, routing, testing, dependencies, deploy.

Cross-stack:

- [project-structure.md](./project-structure.md) — top-level layout of a full project (backend, frontend, infra).

A stack not listed here has no documented conventions — report that and stop.
